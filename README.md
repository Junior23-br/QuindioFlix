# 🎬 QuindioFlix

Proyecto académico **QuindioFlix**, una plataforma de streaming de contenido multimedia desarrollada para el curso **Bases de Datos II**.

El proyecto se centra en el diseño e implementación de una **base de datos Oracle**, aplicando modelado relacional, restricciones de integridad, consultas avanzadas, PL/SQL, transacciones, índices, roles y administración de acceso.

---

## 📌 Descripción general

QuindioFlix simula una plataforma de streaming que permite gestionar usuarios, planes de suscripción, perfiles, contenido multimedia, reproducciones, pagos, calificaciones, favoritos, reportes de contenido y administración de acceso a la base de datos.

---

## 🚀 Tecnologías utilizadas

- 🗄️ Oracle Database
- 🧠 PL/SQL
- 🌿 Git
- 🐙 GitHub

---

## 🎯 Objetivo del proyecto

Construir una solución de base de datos robusta para una plataforma de streaming, permitiendo demostrar los principales temas vistos en el curso de **Bases de Datos II**.

El sistema permite trabajar con:

- 👤 Usuarios, perfiles y planes.
- 🎞️ Catálogo de contenido multimedia.
- ▶️ Reproducciones por perfil y dispositivo.
- ⭐ Calificaciones, reseñas y favoritos.
- 💳 Pagos y estados de suscripción.
- 📊 Reportes y consultas avanzadas.
- 🧠 Procedimientos, funciones, cursores y triggers en PL/SQL.
- 🔁 Transacciones y concurrencia.
- 📈 Índices y análisis con `EXPLAIN PLAN`.
- 🔐 Roles, usuarios, privilegios y perfiles de recursos.
